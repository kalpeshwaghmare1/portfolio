## Running Locally

### Clone the Repository

```bash
git clone https://github.com/kalpeshwaghmare1/Portfolio.git
```

### Navigate to the Project

```bash
cd Portfolio
```

### Install Dependencies

```bash
npm install
```

### Start the Development Server

```bash
npm run dev
```

Visit:

```bash
http://localhost:3000
```
